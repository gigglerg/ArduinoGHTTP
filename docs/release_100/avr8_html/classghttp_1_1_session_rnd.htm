Convert an index from the acceptable session characters alphabet into ASCII.

Note
Use SessionRnd::AlphabetLength to get length
Attention
We cannot use the BASE64 alphabet here as it contains "/" and "+" which are unacceptable for simple inclusion into URLs
Parameters
[in]iIndex into session alphabet. If larger than alphabet, character returned will be length of alphabet (integer)
Returns
ASCII character or length (integer)
