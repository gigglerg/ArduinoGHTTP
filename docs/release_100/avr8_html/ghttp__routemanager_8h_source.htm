47  * Register route. The new route can be a normal route, the last normal route could accept any route to handle the condition where
48  * no other route accepted request and finally it could be a pre-route handler. A pre-route handler makes sure semantics of the request
49  * are generally what your application accepts and can setup other globals to aid your other requests.

51  * \param[in] r Route pointer Make sure your routes are global, don't create them on stack

53  * \return State
54  * \retval +n Route count after addition
55  * \retval -1 Failed. Make sure N is larger enough for your application
56  */
57  int16_t Register(R *r, const bool pre_route_check=false) {
58  bool done = false;
59 
60  if (r) {
