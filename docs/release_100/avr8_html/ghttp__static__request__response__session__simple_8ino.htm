Detailed Description

Example use of GHTTP (Embedded library for parsing HTTP requests and creating responses) - Example of complete request and response using static request data which could have come from any hardware. Session variables are used with a counter which is incremented per request and output in resulting response. A few iterations of request/response are made to demonstrate session handling within your route classes
